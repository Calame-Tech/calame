# Calame — Présentation du produit

Calame est une plateforme qui génère des serveurs MCP (Model Context Protocol)
à partir de bases de données.

## Tarification

La version Pro est facturée à l'instance, à un tarif forfaitaire.
La version Enterprise est sur devis et inclut le support prioritaire.

## Cœur open source

Le cœur de la plateforme est distribué sous licence Apache 2.0 et reste gratuit.
Les fonctionnalités avancées (SSO, RAG) sont publiées sous licence BUSL-1.1.
